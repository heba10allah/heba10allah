## Featured Projects

### 🌱 Agri Tech — AI-Powered Smart Agriculture Platform
**DEPI Graduation Project &nbsp;|&nbsp; Team Project**

> Agri Tech is a real-world AI-powered smart agriculture platform developed as a **team graduation project** for the Digital Egypt Pioneers Initiative (DEPI). The platform supports farmers through intelligent, data-driven decision-making by combining **IoT sensor data**, **Machine Learning**, **Deep Learning**, and **Computer Vision** into five integrated modules, deployed via Streamlit.

<table width="100%">
<tr>
<td width="50%" valign="top">

**Integrated Modules:**
- 🌾 &nbsp;**Soil Analysis** — soil nutrient and condition monitoring using IoT sensor input
- 🌿 &nbsp;**Crop Recommendation Model** — ML model matching optimal crop types to soil profile and environmental conditions
- 💧 &nbsp;**Smart Irrigation Prediction Model** — regression-based model for irrigation scheduling and water optimization

</td>
<td width="50%" valign="top">

**Integrated Modules (cont.):**
- 🔬 &nbsp;**Plant Disease Detection Model** — CNN-based Computer Vision model that identifies crop diseases from leaf images
- 💊 &nbsp;**Treatment Recommendation System** — rule-based and model-assisted prescription of treatment protocols based on detected disease type

</td>
